# Jekyll 写博客真的太简单了

今天我又学会了怎么发布第二篇博客，原来只需要三步：

## 1. 新建文件
在 `_posts` 文件夹里创建一个新文件，文件名必须是：
`年-月-日-标题.md`

## 2. 写内容
用 Markdown 写，和记事本差不多：
- `#` 是大标题
- `##` 是小标题
- `-` 是列表
- `**加粗**` 可以突出重点

## 3. 提交发布
写完直接提交，等一分钟刷新就能看到新文章。

## 我的小收获
- 不用装任何软件，浏览器就能写
- GitHub 自动帮我生成网页
- 免费、稳定、还能随便改内容

以后我可以把学习笔记、代码总结、日常心得都写在这里啦！
